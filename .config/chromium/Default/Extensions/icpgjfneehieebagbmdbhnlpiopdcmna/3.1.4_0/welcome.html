New Tab Redirect is an extension for Google Chrome which will allow you to open any URL when creating a new tab. That's pretty cool for pages on the internet. But... you can also specify a local file like a custom *.html file or a text file!

New Tab Redirect now provides a default new tab experience. Many users rely on chrome://apps/, but also wanted the address bar to be clear and focused for quick search. Others disliked some features of Chrome's original New Tab page like the 'Most Visited Sites' or 'Bookmarks Bar'. New Tab Redirect now offers these features with an option to hide or show them. You can choose to show between 5-20 top sites and between 5-40 bookmarks. If this is the behavior you want, you don't even have to visit the options page. Just create a new tab and check it out!

Click the arrows on the sides of this page (or press the right arrow key) for an introduction to the extension.

If you want to skip the intro, that's cool. You can press CTRL+T or click the new tab button at the top of the browser. Or, you can click here to access the options page.

FAQ

How do I access this extension?

In the top-right of your Google Chrome browser, click the settings icon:  , then select Tools -> Extensions. A new tab will appear. Alternatively, you can type chrome://extensions/ into the address bar.

Scroll down to New Tab Redirect! and click Options.

How do I uninstall this extension?

Follow the directions above, but choose Uninstall.
